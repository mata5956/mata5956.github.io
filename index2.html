Kernaim Menu Not Clickable – Fix

100% Working Fix Steps:

  1. Ensure Proper Display Mode:
    Set the game to Windowed Borderless, not Fullscreen.
  2. Disable Windows Defender Completely:
    Follow both Step 1 & 2 from this guide: Disable Defender Guide
    Use Defender Control Tool to help.
  3. Turn Off Overlays:
    Disable Discord overlay, Steam overlay, and NVIDIA/AMD overlays (via respective apps).
  4. Turn Off System Features:
    • GPU Acceleration
    • Xbox Game Bar
    • Windows Game Mode
    • Core Isolation > Memory Integrity
    • Secure Boot (via BIOS)
    • Fast Startup (Disable Fast Startup)
  5. Update Drivers & Windows:
    Update GPU drivers.
    Make sure Windows is fully updated.

Finally: Reboot your PC after doing all of the above and try launching the loader again.
