How to check if disk format is GPT or MBR:

Open cmd prompt as admin.
Type in diskpart.
Then type in list disk.
If there is an Asterisk (*) below GPT it means that the format is GPT, if empty it means MBR.
(This is to see if the user can switch to UEFI mode.)

How to force enable virtualization - by the legend klaus

1) Make sure virtualization is enabled in BIOS.
2) Press Win + R and then type in msinfo32 and look at the virtualization state.

3) Win + R, type in gpedit.msc, go to Computer config → Administrative templates → System → Device Guard → Double click on "Turn on virtualization based security" and enable it.

4) Open regedit (as admin) → HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\DeviceGuard, double click "EnableVirtualization" and set the value to 1.

5) Restart PC and everything should work.

Kernaim Menu Not Clickable – Fix

100% Working Fix Steps:

  1. Ensure Proper Display Mode:
    Set the game to Windowed Borderless, not Fullscreen.
  2. Disable Windows Defender Completely:
    Follow both Step 1 & 2 from this guide: Disable Defender Guide
    Use Defender Control Tool to help.
  3. Turn Off Overlays:
    Disable Discord overlay, Steam overlay, and NVIDIA/AMD overlays (via respective apps).
  4. Turn Off System Features:
    • GPU Acceleration
    • Xbox Game Bar
    • Windows Game Mode
    • Core Isolation > Memory Integrity
    • Secure Boot (via BIOS)
    • Fast Startup (Disable Fast Startup)
  5. Update Drivers & Windows:
    Update GPU drivers.
    Make sure Windows is fully updated.

Finally: Reboot your PC after doing all of the above and try launching the loader again.
